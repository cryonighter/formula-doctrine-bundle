# Change Log

All notable changes to this project will be documented in this file.
This project adheres to [Semantic Versioning v2.0.0](http://semver.org/).

## [1.0.3] - 2026-06-16
### Added
- Examples with DQL queries in the formula attribute to [README.md](README.md)

### Changed
- Specified the minimum version of the "cryonighter/formula-doctrine" library to ^1.2.0

## [1.0.2] - 2026-05-21
### Added
- Examples with WHERE, ORDER BY, GROUP BY, and HAVING to [README.md](README.md)

### Changed
- Specified the minimum version of the "cryonighter/formula-doctrine" library to ^1.1.4

## [1.0.1] - 2026-05-09
### Changed
- Specified the minimum version of the "cryonighter/formula-doctrine" library to ^1.1.1 and updated the keywords

## [1.0.0] - 2026-05-01
### Added
- Created a bundle for integrating the "cryonighter/formula-doctrine" library into the Symphony framework
